# Coffe Maker Project

Мобильное приложение **Coffe Maker** — это удобный инструмент для заказа любимых кофейных напитков. С его помощью вы можете быстро и легко заказать кофе, чай или другие напитки из вашей любимой кофейни, не тратя время на ожидание в очереди.
> Состояние: в разработке
